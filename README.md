## Getting Started

### Installation

A. Develop apps:

1. Clone the repo
    ```sh
    git clone https://github.com/rwndy/pokemon-memory-game.git
    ```
2. Install packages
    ```sh
    pnpm install
    ```
3. Start react
    ```sh
    pnpm dev
    ```
